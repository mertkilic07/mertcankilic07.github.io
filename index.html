Project 03
Multiplayer Quiz Game

A real-time multiplayer quiz game built from scratch using raw TCP sockets. The server manages concurrent player connections on separate threads, broadcasts questions to all clients simultaneously, and uses a custom length-prefixed JSON protocol for reliable framing. Scoring rewards both correctness and speed — the first player to answer correctly earns bonus points. Both server and client ship with a Tkinter GUI featuring live scoreboards, player lists, and timestamped activity logs.
